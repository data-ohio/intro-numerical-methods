# Conversion notes

These are notes from the process of converting the source from LaTeX to PreTeXt in summer 2026.
Now that the conversion is finished, they have limited value, but could help with another conversion project.

## Modifications to .tex file
First copy the .tex file and do the following changes in e.g. emacs, using string replacement:

Delete:
* \part{...}
* \hide{...}

Replace X by Y, in this order:
* \chapter{ \section{
* \chapter*{ \section{
* \textsc{Matlab} MATLAB
* Matlab MATLAB
* \begin{lstlisting}[frame=none,numbers=left] STARTCONSOLE
* \begin{lstlisting} STARTPROGRAM
* \end{lstlisting} ENDCONSOLEORPROGRAM
* {list} {enumerate}


## Modifications to raw .xml

Next use the online tool at https://pretext.plus/tryit to *Import LaTeX* and get .xml version partially converted to PreTeXt.

Replace:
* STARTCONSOLE <console prompt=">> " lang="matlab"><input>
* STARTPROGRAM <program language="matlab"><code>
* ENDCONSOLEORPROGRAM </input></console>
  * or </code></program>
