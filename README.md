## Collaborator Instructions

### Resources

See the [PreTeXt documentation](https://pretextbook.org/documentation.html) for links to a variety of resources.
To build and publish, you will need the [PreTeXt command line interface (CLI)](https://pretextbook.org/doc/guide/html/processing-CLI.html).


### Building the book 

Within this directory, run

    pretext build web

To check the result, run

    pretext view web

### Publishing to GitHub Pages

Run

    prextext deploy
